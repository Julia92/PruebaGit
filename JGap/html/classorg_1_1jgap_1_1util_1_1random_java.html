Detailed Description

Implementation of a randomX-compliant class based upon the built-in Java.util.Random generator. Note that since the higher level result composition methods are different from those in the (undocumented) standard library, randomJava results will differ from those of the standard library for a given seed.

Designed and implemented in July 1996 by John Walker, kelvin@fourmilab.ch.
