55  // Now we need to tell the Configuration object how we want our
56  // Chromosomes to be setup. We do that by actually creating a
57  // sample Chromosome and then setting it on the Configuration

60  // values (alleles) of those genes to be integers, which represent
61  // how many coins of that type we have. We therefore use the
62  // IntegerGene class to represent each of the genes. That class

65  // --------------------------------------------------------------
