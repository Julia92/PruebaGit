Detailed Description

The Painted Desert problem from Koza's "Evolution of Emergent Cooperative Behavior using Genetic Programming". The problem is to create the same genetic program for a group of ants that will move three colors of sand into columns of like sand.

int [][] examples.gp.paintedDesert.PaintedDesertProblem.readMap ( String  a_filename) throws Exception
inlineprivate

Reads the map from a file. The first line contains the number of X and Y locations and the number of Ants on the map. A 'b' at a location represents a Black grain of sand. A 's' represents a striped grain of sands. A 'g' represents a grey grain of sand. An 'A' represents an Ant. A 'B' represents an Ant and a black grain of sand at the same location. A 'S' represents an Ant and a striped grain of sand at the same location. A 'G' represents an Ant and a grey grain of sand at the same location.

A side effect of this function is setting the m_ants member variable.

Parameters
a_filenameThe location of the file containing the map information.
Returns
An array of the sand locations read in from the map.
