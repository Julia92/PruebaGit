void examples.gp.paintedDesert.AntMap.finalize ( )
inline

Determine the final position of grains of sand. If an ant is carrying sand and there is no sand at the current position, drop the sand at this position. If there is sand, rubberband the sand back to where the ant picked up the sand. If both locations are full, then pick the nearest open spot

int examples.gp.paintedDesert.AntMap.m_currentAnt
private

Holds the instance of the ant being processed. Assumption that some external single threaded process will loop through all the ants using nextAnt to sequence through the ants. It is expected that the clients of the AntMap can use getAnt() to return the Ant instance at the currentAnt index.
