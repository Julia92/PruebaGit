Detailed Description

This class is a helper to allow testing class BulkFitnessOffsetRemoverin a plainforward way: It only works with NumberGeneinstances and returns the addition of their primitive values as the fitness value. This way we can configure a Chromosome with numerical value genes knowing their fitness and see what fitness values should be installed by the method BulkFitnessOffsetRemover.

Do not copy or use this implementation it is for this test only. It is tested itself by the outer class (testDummyFitnessFunction()) to ensure valid test results.
